Current Projects

  • AeroHelper is a helper program that I created to assist me in development of Orionis. The program loads a sprite and allows the user to create named data sets (currently can only handle integers). The user can select a data set and then click the sprite, and the location of the click will be added to the data set. Data pairs can be manually added, removed, moved, or edited after the fact. The data sets can be exported to a text file, which is formatted in such a way as to be copied into a JSON file.

  • Orionis is a 2D space-themed real time strategy game where you - the player - faces off against a computer controlled opponent. You are in direct control of a flagship, a unique and powerful asset in your fleet. From this flagship you request reinforcements and give your fleet orders and can watch from afar as they carry them out, or jump right into the fray directly. Beware however, if your flagship is destroyed then the battle is almost lost. You might get a second chance, if the rest of your fleet can hold out long enough.

    • Orionis | Options is a simple external application that can used to modify the settings that Orionis uses without having to first open Orionis.

The game will be story-driven and will allow the player to choose from a variety of "beginnings" which will change up the story to some degree. I've began some small amount of programming for the game. I am currently using the Godot Engine which is built using C++ and uses a custom built-in language similar to Python.
